Secret Commands

Try typing these in the search box on the Navigate tab:

  • xyzzy - Enable debug mode (Adventure game reference)
  • hal - Activate speech synthesis (2001: A Space Odyssey)
  • hello computer - Alternative speech activation (Star Trek IV)
  • reset - Return to origin
  • help - Show all available commands
  • debug yes/no - Toggle debug mode with parameters
  • hal 1/0 - Toggle speech with numeric boolean

Boolean Expressions

The controller supports LLOGO-style boolean expressions from MIT's 1970s LOGO language:

  • True: yes, y, t, true, right, 1
  • False: no, n, nil, f, false, wrong, 0

These work with any command that accepts parameters!
